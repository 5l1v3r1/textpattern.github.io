bc(code). <txp:if_last_file>


The "if_last_file":/home/www/zendstudio/dokuwiki/bin/doku.php?id=if_last_file tag is a _conditional_ tag and always used as an opening and closing pair, like this...

bc(code). <txp:if_last_file>
...conditional statement...
</txp:if_last_file>


The tag will execute the contained statements if the displayed file is the last in the currently displayed "file_download_list":/home/www/zendstudio/dokuwiki/bin/doku.php?id=file_download_list. The tag supports "else":/home/www/zendstudio/dokuwiki/bin/doku.php?id=else.
