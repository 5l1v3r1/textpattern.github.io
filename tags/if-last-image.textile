bc. <txp:if_last_image>


The "if_last_image":/home/www/zendstudio/dokuwiki/bin/doku.php?id=if_last_image tag is a _conditional_ tag and always used as an opening and closing pair, like this...

bc. <txp:if_last_image>
...conditional statement...
</txp:if_last_image>


The tag will execute the contained statements if the displayed image is the last in the currently displayed "images":/home/www/zendstudio/dokuwiki/bin/doku.php?id=images list. The tag supports "else":/home/www/zendstudio/dokuwiki/bin/doku.php?id=else.
