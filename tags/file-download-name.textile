h1. File download name

h2. Syntax

bc. <txp:file_download_name />

The *file_download_name* tag is a _single_ tag that Textpattern will replace with the name of the file to download. Should be used in a Textpattern 'file' type [[Form template]]. or within a "file_download_link":file-download-link.textile tag.

h2. Attributes

Tag will accept the following attributes (*case-sensitive*):

* @title="boolean"@
Whether to display the file download name or its title.
Values: @0@ (name), or @1@ (title).
Default: @0@.

h2. Examples

h3. Example 1: Display the name of a file, linked to download

bc. <txp:file_download_link filename="my-presentation.pdf">
    <txp:file_download_name />
    [<txp:file_download_size format="mb" decimals="2" />]
</txp:file_download_link>

Other tags used: "file_download_link":file-download-link.textile, "file_download_size":file_download_size.textile.
