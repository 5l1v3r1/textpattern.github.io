h1. Prev title

h2. Syntax

bc. <txp:prev_title />

The *prev_title* tag is a _single_ tag which Textpattern will replace with the name (text) of the previous article in the sort order.

The container tag "link_to_prev":link-to-prev.textile wraps the text or tag and assigns the link.

h2. Attributes

This tag takes no attributes.

h2. Examples

h3. Example 1: Display a link to the previous article when displaying individual articles

bc. <txp:link_to_prev>
    <txp:prev_title />
</txp:link_to_prev>

Other tags used: "link_to_prev":link-to-prev.textile.
