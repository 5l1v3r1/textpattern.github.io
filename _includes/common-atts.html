Common presentational attributes

These attributes, which affect presentation, are shared by many tags.

{% if include.break %}
break="value"
Where value is an HTML element, specified without brackets (e.g. break="li") or some string to separate list items.
Default: {{ include.break }} (see break cross-reference).
{% endif %} {% if include.breakby %}
breakby="integer(s)"
Used to group list items when separating by break. Possible values are lists of integers, like 2 (groups of 2 items) or 1,2 (alternate groups of 1 and 2 items).
Default: {{ include.breakby }} (see breakby cross-reference).
{% endif %} {% if include.class %}
class="class name"
HTML class to apply to the wraptag attribute value, if set, otherwise to the tag itself.
Default: {{ include.class }} (see class cross-reference).
{% endif %} {% if include.escape %}
escape"entities" Global v4.7.0+
Escape HTML entities or other tags.
Default: {{ include.escape}} (see escape cross-reference and tag escaping for details).
{% endif %} {% if include.label %}
label="text"
Label prepended to item.
Default: {{ include.label }} (see label cross-reference).
{% endif %} {% if include.labeltag %}
labeltag="element"
HTML element to wrap (markup) label, specified without brackets (e.g. labeltag="h3").
Default: {{ include.labeltag }} (see labeltag cross-reference).
