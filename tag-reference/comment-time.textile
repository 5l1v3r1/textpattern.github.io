h1. Comment time

h2. Syntax

bc. <txp:comment_time />

The @comment_time@ tag is a _single_ tag which is used to display the time and date the comment was submitted. Used in a comment form.

h2. Attributes

This tag will accept the following attributes (*case-sensitive*):

; [[Attributes Cross-reference#format|format]]="value"
: Override default date format, as set in preferences.
: Values: any valid [http://php.net/strftime strftime()] string or <code>since</code>.
; [[Attributes Cross-reference#gmt|gmt]]="boolean"
: Return either local time -- according to the set time zone preferences -- or GMT.
: Values: <code>0</code> (local time) or <code>1</code> (GMT).
: Default: <code>0</code>.
; [[Attributes Cross-reference#lang|lang]]="ISO language code"
: Format time string suitable for the specified language (locale).
: Values: locales adhere to [http://www.loc.gov/standards/iso639-2/php/code_list.php ISO-639].
: Default: unset (time format set via preferences).

h2. Examples

h3. Example 1: Comments Display Form

bc.	<txp:comment_message />
    <p>
    	<small>&#8212; <txp:comment_name /> <txp:comment_time /> 
		<txp:comment_permlink>#</txp:comment_permlink></small>
	</p>

Other tags used: "comment_name":comment-name.textile, "comment_message":comment-message.textile, "comment_time":comment_time.textile
