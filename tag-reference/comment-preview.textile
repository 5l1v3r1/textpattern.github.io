h1. Comment preview

h2. Syntax

bc. <txp:comment_preview />

The @comment_preview@ tag is a _single_ tag which is used to display a Preview button the user can use to preview the comment text. Used in the comment input form.

h2. Attributes

This tag has no attributes.
