h1. Comment web input

h2. Syntax

bc. <txp:comment_web_input />

The @comment_web_input@ tag is a _single_ tag which is used to display a text entry field to accept the commenter's domain name. This tag can be used in page or form.

Function assumes @http://@ for all URLs.

h2. Attributes

This tag has no attributes.
