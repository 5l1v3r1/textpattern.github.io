Many suggestions for new Textpattern features are posted on the "Textpattern forum":http://forum.textpattern.com/viewforum.php?id=2. The following pages are placeholders for posting more detailed information about potential new features: specifications, documentation, examples, descriptions of tags, screen designs, use cases, and so on. The more detailed and complete the information posted here, the more likely it is that others will contribute.

Before drafting a request, decide if it should be a native core feature of Textpattern, or if it can be a plugin. See "plugin_vs_core":/home/www/zendstudio/dokuwiki/bin/doku.php?id=plugin_vs_core.
