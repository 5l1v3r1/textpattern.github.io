h4(#classification). Classification

The

bc. zem_event_url


tag can be used as a "Single_Tag":/home/www/zendstudio/dokuwiki/bin/doku.php?id=glossary#single_tag or a "Container Tag":/home/www/zendstudio/dokuwiki/bin/doku.php?id=glossary#container_tag. When used as a "Container Tag":/home/www/zendstudio/dokuwiki/bin/doku.php?id=glossary#container_tag the contained text is used as the linktext. A linktext attribute can be used when the tags usage is "Single_Tag":/home/www/zendstudio/dokuwiki/bin/doku.php?id=glossary#single_tag. Returns the value set in the event url field.

h4(#syntax). Syntax

The _zem_event_url_ tag has the following syntactic structure...

bc. <txp:zem_event_url /> or 
<txp:zem_event_url>... contained text ...</txp:zem_event_url>
