\nThese are the main events and steps that Textpattern uses as you interact with the interface, correct as of Textpattern "txp_current_version":/home/www/zendstudio/dokuwiki/bin/lib/exe/fetch.php?id=&media=txp_current_version. Although they are not direct callbacks (see the "core_callback_list":/home/www/zendstudio/dokuwiki/bin/doku.php?id=core_callback_list for those) you may hook into these events and steps from a plugin to perform an action.

Raise interest using the same

bc(code). register_callback


function as regular callbacks, and choose at what stage to hook; event and optional step level, plus whether to run before (pre) Textpattern executes the process, or after it has completed. If

bc(code). pre


is unset, it is assumed 0, i.e. run afterwards:

bc(code). register_callback('event', 'step', pre='boolean')
