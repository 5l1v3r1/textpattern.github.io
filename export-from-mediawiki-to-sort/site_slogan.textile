The "site&#95;slogan":/home/www/zendstudio/dokuwiki/bin/doku.php?id=site_slogan is a _single_ tag which is used to output the site's tagline (labeled as _Site tagline_ in the "basic&#95;preferences":/home/www/zendstudio/dokuwiki/bin/doku.php?id=basic_preferences).

The slogan is a brief (255 characters max) tagline or description of your site which can be used, for example, in XML feeds.

h3(sectionedit1#attributes). Attributes

This tag has no attributes.

h3(sectionedit2#examples). Examples

h4(#example_1general_display_of_slogan). Example 1: General display of slogan

bc(code). <p><txp:site_slogan /></p>


h4(#example_2as_content_filler). Example 2: As content filler

The slogan could be used for the content attribute of the description metadata element. Either whole...

bc(code). <meta name="description" content="<txp:site_slogan />" />


or partial...

bc(code). <meta name="description" content="<txp:site_slogan />. And the rest of your pithy description would go here." />
