bc(code). <txp:if_article_image>


The "if&#95;article&#95;image":/home/www/zendstudio/dokuwiki/bin/doku.php?id=if_article_image tag is a _conditional_ tag and always used as an opening and closing pair, like this...

bc(code). <txp:if_article_image>
...conditional statement...
</txp:if_article_image>


The tag will execute the contained statements if an image is associated (through the "article image field":/home/www/zendstudio/dokuwiki/bin/doku.php?id=write#article_image) with the article being displayed.

h3(sectionedit1#attributes). Attributes

This tag has no attributes.

h3(sectionedit2#examples). Examples

h4(#example_1display_default_image_if_no_article_image_exists). Example 1: Display default image if no article image exists

bc(code). <txp:if_article_image>
    <txp:article_image />
<txp:else />
    <txp:image id="5" />
</txp:if_article_image>
