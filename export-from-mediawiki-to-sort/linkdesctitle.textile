The "linkdesctitle":/home/www/zendstudio/dokuwiki/bin/doku.php?id=linkdesctitle tag is a _single_ tag which is used to return an HTML hyperlink, defined under the "links":/home/www/zendstudio/dokuwiki/bin/doku.php?id=links tab.

It uses the *Title* field as the link's text; the *Description* field contents will be displayed as an anchor title attribute. This tag is used in a link form or inside the "linklist":/home/www/zendstudio/dokuwiki/bin/doku.php?id=linklist container tag.
