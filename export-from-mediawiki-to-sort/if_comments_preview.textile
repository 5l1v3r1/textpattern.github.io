The "if&#95;comments&#95;preview":/home/www/zendstudio/dokuwiki/bin/doku.php?id=if_comments_preview tag is a _conditional_ tag and always used as an opening and closing pair, like this...

bc(code). <txp:if_comments_preview>
...conditional statement...
</txp:if_comments_preview>


The tag will execute the contained statements if a comment is being previewed.
