The "link&#95;to&#95;home":/home/www/zendstudio/dokuwiki/bin/doku.php?id=link_to_home tag is primarily a _container_ tag that returns a link to the site's home page. It will apply a hyperlink to whatever it wraps.

The tag can, however, be used as a _single_ tag to generate a raw base URL of the site:

bc(code). <txp:link_to_home />
