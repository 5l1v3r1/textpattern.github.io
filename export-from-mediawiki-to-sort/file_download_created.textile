The "file&#95;download&#95;created":/home/www/zendstudio/dokuwiki/bin/doku.php?id=file_download_created tag is a _single_ tag that Textpattern will replace with the upload date of the file to download. Should be used in a "download form":/home/www/zendstudio/dokuwiki/bin/doku.php?id=forms_explained.

h3(sectionedit1#attributes). Attributes

Tag will accept the following attributes (*case-sensitive*):

*"format":/home/www/zendstudio/dokuwiki/bin/doku.php?id=attributes_cross-reference#format=“format string”*
 Override the default Archive date format set in the "basic&#95;preferences":/home/www/zendstudio/dokuwiki/bin/doku.php?id=basic_preferences. Values: any valid "strftime()":http://php.net/strftime string values.
