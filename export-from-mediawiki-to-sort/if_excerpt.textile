The "if&#95;excerpt":/home/www/zendstudio/dokuwiki/bin/doku.php?id=if_excerpt tag is a _conditional_ tag and always used as an opening and closing pair, like this...

bc(code). <txp:if_excerpt>
...conditional statement...
</txp:if_excerpt>


The tag will execute the contained statements if an excerpt is associated with the article being displayed.

h3(sectionedit1#attributes). Attributes

This tag has no attributes.

h3(sectionedit2#examples). Examples

h4(#example_1display_the_excerpt_if_it_exists). Example 1: Display the excerpt if it exists

bc(code). <txp:if_excerpt>
  <txp:excerpt />
<txp:else />
  Section <a href="/subdirname/index.php?s=<txp:section />"><txp:section /></a>
</txp:if_excerpt>


If there is no excerpt associated with the current article, the section name linked as */“subdirname/index.php?s=section&#95;name”* will be displayed instead, providing a means of displaying the articles from the same section as the current article.
