The "link&#95;name":/home/www/zendstudio/dokuwiki/bin/doku.php?id=link_name tag is a _single_ tag which returns the name of the link as assigned on the links pane as text. This tag is used in a "links":/home/www/zendstudio/dokuwiki/bin/doku.php?id=links form or inside the "linklist":/home/www/zendstudio/dokuwiki/bin/doku.php?id=linklist container tag to display information about the current link.

h3(sectionedit1#attributes). Attributes

Tag will accept the following attributes (*case-sensitive*):

*"escape":/home/www/zendstudio/dokuwiki/bin/doku.php?id=attributes_cross-reference#escape=“html”*
 Escape HTML entities in link name text. Values:

bc(code). html


or unset. Default:

bc(code). html


.
