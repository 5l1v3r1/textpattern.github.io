The "if&#95;article&#95;list":/home/www/zendstudio/dokuwiki/bin/doku.php?id=if_article_list tag is a _conditional_ tag and always used as an opening and closing pair, like this...

bc(code). <txp:if_article_list>
...conditional statement...
</txp:if_article_list>


The tag will execute the contained statement if an article list is being displayed (i.e. not showing an individual article).

h3(sectionedit1#attributes). Attributes

This tag has no attributes.
