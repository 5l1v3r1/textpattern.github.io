The "link&#95;id":/home/www/zendstudio/dokuwiki/bin/doku.php?id=link_id tag is a _single_ tag which returns the numerical ID of the link. This tag is used in a "links":/home/www/zendstudio/dokuwiki/bin/doku.php?id=links form or inside the "linklist":/home/www/zendstudio/dokuwiki/bin/doku.php?id=linklist container tag to show information about the current link in the list.

h3(sectionedit1#attributes). Attributes

This tag has no attributes.
