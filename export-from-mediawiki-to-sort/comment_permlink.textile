The "comment&#95;permlink":/home/www/zendstudio/dokuwiki/bin/doku.php?id=comment_permlink tag is a _container_ tag which is used to return the permanent link of the article comment being displayed. The container tag wraps the text assigned to the link.

h3(sectionedit1#attributes). Attributes

Tag will accept the following attributes (*case-sensitive*):

*"anchor":/home/www/zendstudio/dokuwiki/bin/doku.php?id=attributes_cross-reference#anchor=“boolean”*
 Whether to apply the comment's id to the hyperlink tag (as the id attribute), setting this comment permanent link as the comment page anchor. Values:

bc(code). 1


(yes) or

bc(code). 0


(no). Default is

bc(code). 0


.
