The "file&#95;download&#95;id":/home/www/zendstudio/dokuwiki/bin/doku.php?id=file_download_id&gt; tag is a _single_ tag that Textpattern will replace with the internal ID number of the file to be downloaded. Should be used in a "download form":/home/www/zendstudio/dokuwiki/bin/doku.php?id=forms_explained.

h3(sectionedit1#attributes). Attributes

This tag has no attributes.
