The "search&#95;term":/home/www/zendstudio/dokuwiki/bin/doku.php?id=search_term tag is a _single_ tag which returns the expression the user searched for through the full text search form.

h3(sectionedit1#attributes). Attributes

Tag will accept the following attributes (*case-sensitive*):

*_align_[^deprecated^]=“html”*
 When set to

bc(code). html


, HTML entities are used for &lt;, &gt;, and &amp; in the output text. As of Textpattern 4.5 the attribute is deprecated and its use is highly unrecommended. Default:

bc(code). html
