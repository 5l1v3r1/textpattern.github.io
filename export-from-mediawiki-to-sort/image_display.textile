The "image&#95;display":/home/www/zendstudio/dokuwiki/bin/doku.php?id=image_display tag is a _single_ tag that is intended to be used in tandem with "image&#95;index":/home/www/zendstudio/dokuwiki/bin/doku.php?id=image_index.

The "image&#95;display":/home/www/zendstudio/dokuwiki/bin/doku.php?id=image_display tag displays an image specified by the page URL which in turn is built by its tandem tag "image&#95;index":/home/www/zendstudio/dokuwiki/bin/doku.php?id=image_index.

To use this tag successfully, it has to be placed either inside an article which shares a common category with the images to display (thereby linking article and image categories), or in a location at the page template which is displayed without any special article context.

If this tag seems to display no image at all, it probably resides inside an article which is never rendered as it does not belong to the currently active category.

h3(sectionedit1#attributes). Attributes

This tag takes no attributes.
