h4(#classification). Classification

The

bc(code). zem_event_description


tag is used as a "Single&#95;Tag":/home/www/zendstudio/dokuwiki/bin/doku.php?id=glossary#single_tag. Used in a zem&#95;event&#95;list form, it returns the contents of the event's decription field..

h4(#syntax). Syntax

The _zem&#95;event&#95;description_ tag has the following syntactic structure...

bc(code). <txp:zem_event_description />


h4(#attributes). Attributes

Tag will accept the following attributes (note: attributes are *case sensitive*):

*&lt;code&gt;*escape=“*_0 or 1_*“*&lt;/code&gt;*
 html escape. Default is

bc(code). 0


. *&lt;code&gt;*field=“*_field name(internal use)_*“*&lt;/code&gt;*
 Default is

bc(code). description_html


. *&lt;code&gt;*wraptag=“*_wraptag text_*“*&lt;/code&gt;*
 HTML tag to be used as the wraptag, without brackets. Default is “div”. *&lt;code&gt;*class=“*_class name_*“*&lt;/code&gt;*
 CSS class attribute, default is

bc(code). **zem_event_description**


.
