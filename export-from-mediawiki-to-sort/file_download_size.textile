The "file&#95;download&#95;size":/home/www/zendstudio/dokuwiki/bin/doku.php?id=file_download_size tag is a _single_ tag which Textpattern will replace with the formatted file size of the file to be downloaded. Should be used in a "download form":/home/www/zendstudio/dokuwiki/bin/doku.php?id=forms_explained.

h3(sectionedit1#attributes). Attributes

Tag will accept the following attributes (*case-sensitive*):

*"decimals":/home/www/zendstudio/dokuwiki/bin/doku.php?id=attributes_cross-reference#decimals=“places”*
 Number of decimal places to format the value to. Default:

bc(code). 2


. *"format":/home/www/zendstudio/dokuwiki/bin/doku.php?id=attributes_cross-reference#format=“numbering style”*
 The way to represent the number, based on the file's expected size. Valid options are: b (bytes) k (kilobytes) m (megabytes) g (gigabytes) t (terabytes) p (petabytes) e (exabytes) z (zettabytes) y (yottabytes) Default: unset (i.e. the most appropriate units based on the file size)

h3(sectionedit2#examples). Examples

h4(#example_1display_formatted_file_size_in_kilobytes). Example 1: Display formatted file size in kilobytes

bc(code). <txp:file_download_size format="k" />
