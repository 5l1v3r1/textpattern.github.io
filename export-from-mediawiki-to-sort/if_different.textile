The "if&#95;different":/home/www/zendstudio/dokuwiki/bin/doku.php?id=if_different tag is a _conditional_ tag and always used as an opening and closing pair, like this...

bc(code). <txp:if_different>
...conditional statement with <txp:other_tag>...
</txp:if_different>


The tag will execute the contained statement when the value of the contained statement differs from the preceding value for that contained statement. Can be used in article, link, comment, and file forms.

*if&#95;different* can contain several HTML tags but only _one_ Textpattern tag.

You can "work around this limitation":http://txptips.com/extend-if_different-using-the-variable-and-if_variable-tags by means of the use of the "variable":/home/www/zendstudio/dokuwiki/bin/doku.php?id=variable and "if&#95;variable":/home/www/zendstudio/dokuwiki/bin/doku.php?id=if_variable tags.
