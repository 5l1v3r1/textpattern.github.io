h3(sectionedit2#examples). Examples

h4(#example_1display_some_localized_text). Example 1: Display some localized text

bc(code). <txp:older><txp:text item="older" /></txp:older>


*What it does...*
 Outputs the text 'older' inside the &lt;txp:older /&gt; tag, respecting the current Textpattern language.

*Why you might use this...*
 Instead of using the tag like this:

bc(code). <txp:older>older</txp:older>


which would always render the English text 'older', it replaces the contents with the value assigned to the name 'older' in the current language. So you would see a link with the word 'Ã¤lter' if you were using German (de) as the Textpattern site language.
