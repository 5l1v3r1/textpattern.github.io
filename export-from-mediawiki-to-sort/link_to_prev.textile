The "link&#95;to&#95;prev":/home/www/zendstudio/dokuwiki/bin/doku.php?id=link_to_prev tag can be used as a _single_ tag or a _container_ tag to return the permanent URL of the previous article by posting date.

If used as a container tag, the HTML required to output a hyperlink is returned; if used as a single tag, only the URL itself is returned.

h3(sectionedit1#attributes). Attributes

Tag will accept the following attributes (*case-sensitive*):

*"showalways":/home/www/zendstudio/dokuwiki/bin/doku.php?id=attributes_cross-reference#showalways=“boolean”*
 Show the wrapped value even when no previous article exists. Values:

bc(code). 0


(no) or

bc(code). 1


(yes). Default:

bc(code). 0


h3(sectionedit2#examples). Examples

h4(#example_1link_to_previous_article_using_its_title). Example 1: Link to previous article using its title

Note: while

will enable this tag to display what is wrapped inside it, "prev&#95;title":/home/www/zendstudio/dokuwiki/bin/doku.php?id=prev_title returns nothing if there is no previous title, so nothing is displayed. Use text or the returned value that you need displayed.

h4(#example_3customizing_links). Example 3: Customizing links

The container tag returns only a very basic link, which doesn't allow for customizing the link title, or adding a CSS class, etc. Using the tag in its single tag capacity opens up a lot more possibilities.

For example, to give the link an HTML title attribute of the previous article's title, and also apply a class to it:

bc(code). <a href="<txp:link_to_prev />" title="<txp:prev_title />" class="orange"><txp:prev_title /></a>
