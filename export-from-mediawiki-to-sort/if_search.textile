The "if&#95;search":/home/www/zendstudio/dokuwiki/bin/doku.php?id=if_search tag is a _conditional_ tag and always used as an opening and closing pair, like this...

bc(code). <txp:if_search>
...conditional statement...
</txp:if_search>


The tag will execute the contained statements if the called page is the result of a search.

h3(sectionedit1#attributes). Attributes

This tag has no attributes.

h3(sectionedit2#examples). Examples

h4(#example_1select_a_different_stylesheet_during_search). Example 1: Select a different stylesheet during search

bc(code). <txp:if_search>
  <link rel="stylesheet" href="<txp:css name="search" />" type="text/css" />
<txp:else />
  <link rel="stylesheet" href="<txp:css />" type="text/css" />
</txp:if_search>


*What this does...*
 Selects a stylesheet named “search” when search results are being displayed, or a stylesheet determined by the active section for normal page display.
