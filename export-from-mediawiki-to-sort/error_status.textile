The "error&#95;status":/home/www/zendstudio/dokuwiki/bin/doku.php?id=error_status tag is a _single_ tag that Textpattern will replace with the error status as set by the server. Should be used in an error&#95;xxx or error&#95;default "page template":/home/www/zendstudio/dokuwiki/bin/doku.php?id=pages.

h3(sectionedit1#attributes). Attributes

This tag takes no attributes.

h3(sectionedit2#examples). Examples

h4(#example_1display_error_information). Example 1: Display error information

bc(code). <h3><txp:error_status /></h3>
<p><txp:error_message /></p>


*What this does...*
 With the tags arranged like this (as they are in the error&#95;default page template), they display the error status code as a header and the relevant server message beneath it, usually to indicate to the visitor that something went wrong.
