h2. Filename (URL) conventions

For SEO friendly URLs, and for consistency, all directory names/file names should be structured using the following patterns:

# All text in lowercase.
# Use hyphens, *not* underscores, to separate words within the URL.
# Ensure @.textile@ prefix is used on files (Markdown is not be used).

*Example:*

bc. file-download-description.textile
